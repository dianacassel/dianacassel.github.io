# Diana Cassel

Site pessoal estatico de Diana Cassel, preparado para publicacao via GitHub Pages.

## Estrutura

- `index.html`: pagina inicial
- `about.html`: sobre
- `work.html`: trabalhos
- `writing.html`: textos
- `contact.html`: contato
- `styles.css`: estilos globais
- `lang.js`: alternancia PT/EN

## GitHub Pages
